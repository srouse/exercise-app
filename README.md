# Rest timer (workout)

Minimal React + Vite SPA: between-exercises countdown, alarm, **Done**, session persistence in `localStorage`.

## Scripts

```bash
npm install
npm run dev      # http://localhost:5173 — use Safari for storage / audio checks
npm run build    # output in dist/
npm run preview  # serve production build locally
```

## Spec & QA

Feature design and manual test matrix: `specs/001-bootstrap-session-timer/quickstart.md`
